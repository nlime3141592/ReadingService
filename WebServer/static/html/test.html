POST test 1

POST 요청을 form 태그에서 수행하는 방법을 보여줍니다.

이 버튼은 작동하지 않습니다.

테스트 결론

동작 방식이 복잡하므로 사용하지 않도록 한다.

  1. HTML에서 name 속성을 통해 보낼 데이터가 포함된 컨테이너를 명시
  2. 추가 데이터를 보내려면 제출 버튼을 누를 때 javascript로 form 객체를 얻은 후 type="hidden"인 input 태그를 동적으로 생성해 데이터를 삽입해줘야 함.
위에 따르면 html 파일과 js파일을 동시에 관리해야 하는데 이는 복잡하다.
